# linux-redHat-project
Baseline Hardening &amp; Threat Monitoring on Linux (RHEL-based)



# Obiettivo
Dimostrare competenze pratiche in:
- gestione e configurazione di sistemi Linux (Red Hat-like)
- sicurezza di base del sistema operativo
- gestione utenti, permessi e servizi
- threat detection tramite OSSEC/Wazuh
- scansione vulnerabilità (Lynis)
- gestione log, auditing e firewall
- produzione documentazione tecnica chiara

Il tutto realizzato in una VM Red Hat-based (Fedora, Rocky, Alma o RHEL trial).

# Risultati finali attesi
- Sistema RHEL hardened, con:
- firewall configurato
- auditing attivo
- utenti e permessi corretti
- servizi minimi
- SSH messo in sicurezza
- Wazuh/OSSEC agent configurato e funzionante.
- Report di vulnerabilità con Lynis.
